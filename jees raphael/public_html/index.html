Loop Checking

Loop checking in instrumentation is a process that verifies that an automated system's control loop is working as intended. It's usually performed before the system is commissioned, after all field instruments have been installed.

Academic Projects

Dam Automation system using LabVIEW

Main project

This is a miniature version of an automatic dam shutter control system. As the name suggests, the dam operations are carried out automatically. An ultrasonic sensor continuously monitors the water level. There are three specific levels at which, whenever the water reaches them, the dam shutter will automatically open, and alerts are generated for the respective water levels. The graphical representation and SMS notifications are supported by LabVIEW and a GSM module. As the water level decreases, the dam shutter closes accordingly.

IOT Theft notification system

Mini project

Nowadays, technology is advancing very quickly, and everyone is very concerned about security. This project makes us more secure by letting you know whether a specified door is open or closed, no matter where you are, as long as the device is connected to the internet and your phone has network access. Whenever the gate opens, the magnet separates from the reed switch, breaking the circuit inside. This break in the circuit triggers the NodeMCU to send a status update to the Blynk server, which pushes a notification to your phone about the event. You can receive door status notifications from anywhere in the world via an internet connection.
